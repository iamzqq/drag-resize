# rc-drag-resize

使子元素可以八向拖动改变大小的 React 组件

## 使用方式

```tsx
import React from 'react';
import DragResize from 'rc-drag-resize';

import 'rc-drag-resize/es/index.css';

export default () => (
  <DragResize>
    <div className="resize-element" />
  </DragResize>
);
```
